com.ibm.streamsx.thrift

General Information

Support for thrift protocol.

This toolkit provides thrift server and thrift client functionality. Thrift idl file and thrift c++ code are generated automatically based on SPL type definitions.

The toolkit supports operator port schemas with basic types as well as collections of basic types.

Below a table of the supported basic types: 

thrift	SPL

binary	blob
bool	boolean
byte	int8
i16	int16
i32	int32
i64	int64
double	float64
string	rstring

Third-party libraries

  • This toolkit embeds thrift headers/binaries under impl/include|lib|bin.
